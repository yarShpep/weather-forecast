# Прогнозирование погоды в Москве с использованием LSTM и Transformer

## Описание проекта

Проект посвящён прогнозированию среднесуточной температуры воздуха в Москве на основе архивных данных за последние 10 лет с использованием нейронных сетей.  
Реализованы и сравнены следующие подходы:

- baseline-модели (naive, moving average, seasonal naive);
- рекуррентная нейронная сеть **LSTM**;
- модель **Transformer** с позиционным кодированием.

Основная цель — сравнить качество моделей и построить прогноз на дату защиты курсовой работы.

## Источник данных

Архивные метеорологические данные получены с сайта:  
http://www.pogodaiklimat.ru/monitor.php?id=27612  

Используется метеостанция №27612 (Москва).

## Инструкция по запуску

1. Скачать репозиторий или архив с кодом.
2. Установить зависимости.
3. Открыть файл weather_forecast.ipynb.
4. Запустить все ячейки сверху вниз.

При отсутствии файла data/weather_raw.csv данные будут загружены автоматически.

## Воспроизводимость

- Зафиксированы случайные состояния (random, numpy, torch).
- Сохранены чекпойнты лучших моделей:
  - data/lstm.pt
  - data/transformer.pt
- Реализованы функции загрузки моделей без повторного обучения.

## Метрики качества

Используются MAE и RMSE.

Модель | MAE | RMSE
Naive | 3.122 | 4.137
Moving Avg | 3.373 | 4.379
Seasonal Naive | 4.431 | 5.723
LSTM | 2.849 | 3.637
Transformer | 3.005 | 3.813

## Прогноз на дату защиты

Модели строят прогноз на 7 дней вперёд для выбранной даты запуска.
